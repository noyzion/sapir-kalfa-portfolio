# הסבר מפורט על הפרויקט

## סקירה כללית
בניתי אתר תיק עבודות מודרני לאדריכלית ספיר קלפה, עם תמיכה בעברית ואנגלית, עיצוב מינימליסטי בשחור/אפור/לבן.

---

## מה נבנה?

### 1. **מבנה הפרויקט הבסיסי**
- **React 18** עם **Vite** - סביבת פיתוח מהירה
- **CSS Modules** - כל קומפוננטה עם קובץ CSS משלה
- **Context API** - לניהול מצב השפה

### 2. **מערכת בחירת שפה (i18n)**
**קובץ: `src/context/LanguageContext.jsx`**

מה זה עושה:
- מנהל את השפה הנוכחית (עברית/אנגלית)
- שומר את הבחירה ב-localStorage (נשמר גם אחרי רענון)
- משנה את כיוון הטקסט (RTL/LTR) אוטומטית
- מספק תרגומים לכל הטקסטים באתר

איך זה עובד:
```javascript
// כל קומפוננטה משתמשת ב-hook:
const { t, language, setLanguage } = useLanguage()

// t מכיל את כל התרגומים לפי השפה הנוכחית
// setLanguage משנה את השפה
```

### 3. **קומפוננטות האתר**

#### **Navigation** (תפריט ניווט)
- תפריט קבוע בחלק העליון
- כפתור החלפת שפה (EN/עב)
- תפריט מובייל (המבורגר)
- משנה מראה כשגוללים למטה

#### **Hero** (עמוד בית)
- כותרת גדולה עם שם האדריכלית
- תיאור קצר
- כפתור CTA (Call to Action) לפרויקטים
- עיצוב מינימליסטי עם רווחים גדולים

#### **About** (אודות)
- כותרת גדולה
- שני פסקאות תיאור
- 3 כרטיסי כישורים (תכנון אדריכלי, עיצוב פנים, שיפוץ)

#### **Projects** (פרויקטים)
- גריד של 2 עמודות (1 במובייל)
- 6 פרויקטים לדוגמה
- כל פרויקט: תמונה, כותרת, תיאור, קטגוריה
- אפקט hover עדין

#### **Contact** (צור קשר)
- פרטי קשר (אימייל, טלפון, כתובת)
- טופס יצירת קשר
- שדות עם גבול תחתון מינימליסטי
- כפתור שליחה עם אפקט fill

### 4. **עיצוב וסטיילינג**

#### **פלטת צבעים**
- שחור (#000000) - טקסטים ראשיים
- אפור כהה (#1a1a1a) - טקסטים משניים
- אפור (#4a4a4a) - טקסטים קלים
- אפור בהיר (#8a8a8a) - טקסטים עדינים
- לבן (#ffffff) - רקע

#### **טיפוגרפיה**
- גדלי טקסט רספונסיביים עם `clamp()`
- משקלים דקים (200-300)
- רווחי אותיות מותאמים
- שורות ברווח נוח (line-height: 1.7-1.8)

#### **רווחים ופריסה**
- רווחים גדולים בין סקציות (100px)
- max-width: 1400px לכל הסקציות
- padding אחיד (5% בצדדים)

#### **אפקטים ואנימציות**
- כפתורים עם אפקט fill על hover
- כרטיסי פרויקטים עם hover עדין
- transitions חלקות (0.3s)

### 5. **רספונסיביות**
- **Desktop**: פריסה מלאה עם 2 עמודות לפרויקטים
- **Tablet** (968px): התאמות בינוניות
- **Mobile** (768px): פריסה אנכית, תפריט המבורגר

## טכנולוגיות וכלים

### Dependencies:
- **react** + **react-dom** - ספריית React
- **vite** - Build tool מהיר
- **@vitejs/plugin-react** - פלאגין React ל-Vite

### תכונות מיוחדות:
- **Context API** - ניהול מצב גלובלי (שפה)
- **localStorage** - שמירת בחירת שפה
- **CSS Variables** - ניהול צבעים מרכזי
- **clamp()** - גדלי טקסט רספונסיביים
- **Smooth Scroll** - גלילה חלקה בין סקציות
